# Notebook Guide

Suggested notebooks for presentation and experimentation:

1. `01_eda.ipynb`
   - dataset overview

   - cancellation patterns
2. `02_modeling.ipynb`
   - preprocessing experiments

   - elbow method

The production-ready logic already exists in `src/hotel_ml/`. Use notebooks only for exploration and visualization.

# Project Report Template

## Title

Smart Hotel Revenue Management: Booking Cancellation Prediction and Guest Segmentation

## Problem

Hotels lose revenue when guests cancel reservations, especially close to the check-in date. This project predicts cancellation risk and segments guests into behavior-based groups to support revenue management decisions.

## Objectives

- Compare multiple machine learning models for cancellation prediction
- Identify important booking features
- Segment guests using K-Means clustering
- Deploy results in a Streamlit dashboard

## Methodology

- Data cleaning and missing-value handling
- Feature engineering
- Imbalance handling with SMOTE
- Training and evaluation of multiple models
- Clustering and segment profiling
- Deployment with Streamlit

## Deliverables

- Trained models in `artifacts/`
- Evaluation tables
- Cluster summaries
- Live web app
